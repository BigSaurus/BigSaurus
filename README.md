# Hi, I'm Big Saurus

I build **accessible tools for musicians and audio engineers** — software that makes guitar gear, audio interfaces, and MIDI controllers usable with a screen reader and keyboard, not just a mouse and a pretty UI.

Most of these projects started because a piece of music hardware or software *wasn't* accessible, and I wanted to fix that.

---

## Projects

### Guitar Tuner
A fully accessible guitar tuner built for screen-reader users, with guided tuning and alternate-tuning profiles. Available for Windows, with **iOS and Android versions planned**.
> *Commercial product — purchase link coming soon.*

### Morningstar MC8 Pro Editor (MCP)
Open-source Python MCP server for controlling a Morningstar MC8 Pro foot controller over USB MIDI SysEx — edit presets and banks programmatically.
> **[View on GitHub](https://github.com/BigSaurus/MorningStar-Editor-MCP)**

### Axe-Fx III MCP & SysEx Tooling
Reverse-engineering the Fractal Audio Axe-Fx III SysEx protocol and an MCP server to drive it — block layouts, amp models, and parameter control.
> *Private / in active development.*

### CueMix 5 Accessibility
Accessibility overlay and tooling for MOTU's CueMix 5 mixer app, making the audio-interface routing and monitoring controls screen-reader friendly.
> *Private / in active development.*
